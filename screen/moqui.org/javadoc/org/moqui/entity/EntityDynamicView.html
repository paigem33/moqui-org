Package: org.moqui.entity

[Java] Interface EntityDynamicView

  • This class is used for declaring Dynamic View Entities, to be used and thrown away. A special method exists on the EntityFind to accept a EntityDynamicView instead of an entityName. The methods here return a reference to itself (this) for convenience.
